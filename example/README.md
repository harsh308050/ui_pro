# UI Pro Example App

A comprehensive demonstration of all UI Pro package widgets and components.

## Features Demonstrated

### 1. **Buttons & Actions**
- UIProButton (filled, outlined, text variants)
- Loading states
- Icon buttons
- Custom styling options
- Disabled states

### 2. **Text Fields**
- Basic text input
- Email field with validation
- Password field with visibility toggle
- Phone number input
- Search field with clear button
- Multiline text area
- Labels and hints

### 3. **Navigation**
- Standard Bottom Navigation
- Floating Bottom Navigation  
- Fluid Bottom Navigation (with wave animation)
- UIProAppBar with actions
- UIProDoubleTapExit (tap twice to exit)

### 4. **Lists & States**
- UIProListView with shimmer loading
- Empty state with actions
- Dynamic list items
- Reload functionality

### 5. **Loading & Shimmer**
- Box shimmer
- Circle shimmer
- Shimmer list
- Various loading animations:
  - Wave Dots
  - Horizontal Dots
  - Progressive Dots
  - Hexagon Dots
  - Flickr
- Custom shimmer colors

### 6. **Swipeable Tiles**
- UIProSwipeableTile with delete action
- Swipe gestures
- Undo functionality
- Empty state when all deleted

## Running the Example

```bash
cd example
flutter run
```
